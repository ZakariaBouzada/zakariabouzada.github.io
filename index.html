Projects

Automated Medallion Data Pipeline (Academic Project)

Designed and implemented a fully automated data pipeline on Microsoft Databricks following the medallion architecture (Bronze → Silver → Gold). The pipeline ingests and processes 73+ years of NBA statistics across 22 datasets and over 5,300 unique players, performing structured transformations in Python and Apache Spark. Gold-layer outputs power both a JavaScript dashboard and a Power BI report.

AI Search & LLM Demo

A production-deployed AI search engine integrated into this portfolio. Powered by Meta LLaMA 3.2 (1B parameters) via the Hugging Face Inference API, with a Python Flask middleware layer handling routing, CORS, and API key security. The backend is deployed on Render; the frontend runs asynchronous fetch requests for real-time chat responses. Demonstrates full-stack AI integration from model API to deployed UI.

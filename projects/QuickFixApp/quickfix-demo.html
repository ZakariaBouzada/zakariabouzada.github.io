Technical Architecture

  • Native Android app built with Kotlin, iOS with Swift
  • Firebase handles auth, Realtime Database, and Analytics
  • Node.js middleware secures ImageKit API keys for uploads
  • OkHttp client handles all backend API communication
  • Google Maps SDK powers real-time location features
  • This demo uses Leaflet.js to simulate the core flow

Business Model

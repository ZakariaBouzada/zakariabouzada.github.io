Project Overview

This project involved designing and implementing an automated data pipeline following the medallion architecture on Microsoft Databricks. The goal was to ingest and process NBA player statistics in near real time and produce structured analytical datasets for performance analysis.

Architecture

The pipeline follows the medallion architecture pattern, consisting of Bronze, Silver, and Gold layers. Raw NBA statistics are ingested into the Bronze layer, cleaned and structured in the Silver layer, and aggregated into analytics-ready datasets in the Gold layer.

Technologies Used

  • Microsoft Databricks
  • Python (data ingestion and transformation)
  • SQL (data modeling and aggregations)
  • Delta Lake
  • Power BI (data visualization)

My Role and Contributions

I was responsible for designing the pipeline architecture, implementing data transformations using Python and SQL, and configuring automated jobs within Databricks. Particular emphasis was placed on data quality, scalability, and clear separation of pipeline layers.

Results and Visualization

The final Gold-layer datasets categorized NBA players based on scoring and assist performance. These datasets were used to build interactive dashboards in Power BI, enabling comparative analysis and trend exploration.
